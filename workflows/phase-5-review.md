# Phase 5 — Review and Iterate

**Trigger**: User reads the draft and gives feedback

---

## Steps

1. **Apply corrections as requested**
   - Make the changes the user asks for

2. **Adjust tone if needed**:
   - If user says something is off with the tone, adjust and show the corrected paragraph

3. **Iterate**:
   - Repeat until the user says "c'est bon", "publish", or similar
