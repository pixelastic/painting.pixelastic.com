# Phase 4.7 — Flow & Polish

**Input**: Phase 4.5 (Humanization) is complete. The text is natural and human-sounding, but may have choppy sentences, repetitions, or abrupt transitions from the literal speech-to-text translation.

This phase improves readability by merging short sentences, eliminating repetitions, and adding transitions where there's a clear logical connection.

---

## Principles

1. **Merge short choppy sentences** into flowing prose
2. **Eliminate obvious repetitions** of words and phrases
3. **Add transitions** ONLY when there's a clear logical connection between paragraphs
4. **Don't reorganize content** - keep images in the same order
5. **Preserve voice and personality** - don't make it too formal

---

## Step 1: Merge Short Choppy Sentences

Look for sequences of 2-3 very short sentences (5-10 words each) that can be combined into one flowing sentence.

### When to Merge

- Sentences express closely related ideas
- Sentences describe the same subject
- Reading them separately feels choppy

### How to Merge

Use conjunctions, commas, or relative clauses:

**Bad** (choppy):
```
It takes up a lot of space. I can't see what's in the bins. The bins can be too small.
```

**Good** (merged):
```
It took up a lot of space, the bins were often too small, and I couldn't easily see what was inside.
```

**Bad** (choppy):
```
I can't remember what they're called. They're from Eastern Europe. I don't remember exactly where.
```

**Good** (merged):
```
I can't remember what they're called or exactly where they're from, somewhere in Eastern Europe.
```

### When NOT to Merge

- Don't merge if it creates a sentence longer than 30-35 words
- Don't merge if the ideas are distinct and deserve their own sentences
- Don't merge if it loses impact (sometimes short = punchy)

---

## Step 2: Eliminate Repetitions

Scan for words or phrases repeated multiple times in the same paragraph or across consecutive paragraphs.

### Common Repetitions

- Same word at the start of multiple sentences ("I had... I had... I had...")
- Same phrase repeated ("I don't remember... I don't remember...")
- Same verb used 3+ times in one paragraph

### How to Fix

**Option A: Merge sentences** (if they're closely related)

**Bad**:
```
I realize how small my work surface is. I realize the storage isn't optimized.
```

**Good**:
```
I realize how small my work surface is and how poorly optimized the storage is.
```

**Option B: Use pronouns or synonyms**

**Bad**:
```
I had furniture with plastic drawers. I had furniture for storing screws.
```

**Good**:
```
I had furniture with plastic drawers, the kind normally used for storing screws and nails.
```

**Option C: Remove redundant clarifications**

**Bad**:
```
I can't remember what they're called, from Eastern Europe. I don't remember exactly where it's located.
```

**Good**:
```
I can't remember the brand name, but they're from somewhere in Eastern Europe.
```

### When NOT to Fix

- Don't remove repetition if it's intentional for emphasis ("heaps and heaps")
- Don't change technical terms that need to be precise
- Don't eliminate repetition if it hurts clarity

## Step 3: Add Transitions (Only When Clear Connection Exists)

Review paragraph transitions. Add a transition ONLY if there's an obvious logical connection:

### Type 1: Problem → Solution

**Before**:
```
(Image 11) ...it's hard to find exactly what you want.

(Image 12) The large drawers that are the most practical.
```

**After**:
```
(Image 11) ...it's hard to find exactly what you want.

(Image 12) The large drawers turned out to be much more practical.
```

### Type 2: Temporal Progression

**Before**:
```
(Image 1) ...I stopped trying to optimize it.

(Image 2) This is how it was at the start.
```

**After**:
```
(Image 1) ...I stopped trying to optimize it.

(Image 2) But it didn't start that way. At first, I stacked IKEA furniture...
```

### Type 3: Evolution/Change

**Before**:
```
(Image 4) ...my stock isn't optimized at all.

(Image 5) This is after I invested in special boxes.
```

**After**:
```
(Image 4) ...my stock isn't optimized at all.

(Image 5) That's when I decided to invest in special modular boxes...
```

### Transition Words to Use

- **Temporal**: "At first", "Later", "Eventually", "Around this time"
- **Contrast**: "But", "However", "turned out to be"
- **Causation**: "That's when", "So I decided to"
- **Continuation**: "This evolved into", "From there"

### When NOT to Add Transitions

**CRITICAL**: Only add transitions when there's a **clear, obvious logical connection**.

**Don't force transitions** when:
- Images describe completely different subjects
- There's no temporal or causal relationship
- The original abrupt transition works fine

**Example of when NOT to add transition**:
```
(Image 5) I painted a demon miniature with red skin.

(Image 6) This is the base I made for a completely different project.
```
→ No transition needed. They're unrelated topics.

---

## Step 4: Apply Changes

Go through the text **paragraph by paragraph**:

1. Read each paragraph
2. Identify choppy sentences → merge if appropriate
3. Identify repetitions → eliminate if appropriate
4. Check transition to next paragraph → add connector ONLY if clear connection exists
5. Re-read to ensure it still sounds natural

---

## Step 5: Quality Check

After editing, verify:

1. ✅ Does it flow better when reading out loud?
2. ✅ Are obvious repetitions gone?
3. ✅ Do transitions feel natural (not forced)?
4. ✅ Does it still sound like the person talking?
5. ✅ Is the voice still personal and conversational?
6. ❌ Did we make it too formal or literary?
7. ❌ Did we lose any important details?
8. ❌ Did we add transitions where there's no logical connection?

---

## Step 6: Present to User

Tell the user:

**"Voilà le texte après la passe de flow & polish. J'ai fusionné quelques phrases courtes, éliminé les répétitions, et ajouté quelques transitions là où c'était logique."**

Then wait for user feedback.

---

## What NOT to Do

- ❌ Don't reorganize images or content
- ❌ Don't add flowery language or make it literary
- ❌ Don't force transitions between unrelated paragraphs
- ❌ Don't merge sentences if it creates overly long sentences (35+ words)
- ❌ Don't remove personality or conversational tone
- ❌ Don't change technical terms or specific details
